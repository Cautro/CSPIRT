# CSPIRT - Социальный рейтинг

Проект для школы. Система отслеживания "социального рейтинга" учеников (мой первый full-stack проект).

## Стек
- Frontend: React + Vite + TypeScript + React Router
- Backend: Node.js + Express + TypeScript

## Установка и запуск

```bash
git clone https://github.com/Cautro/CSPIRT.git
cd CSPIRT

# Frontend
cd client
npm install
npm run dev  # http://localhost:5173

# Backend (в отдельном терминале)
cd ../server
npm install
npm run dev  # http://localhost:5000
